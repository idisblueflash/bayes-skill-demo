# 更新日誌

記錄 `bayesian-hypothesis-live` 每次版本的重要變更。

## [0.6.0] - 2026-05-18

### 改進
- 貝氏歸一化計算改由 `bayes_calc.py` 執行，確保後驗機率精確且恆等於 100%
- 技能不再依賴模型自行運算，消除偶發的數學錯誤

## [0.5.1] - 2026-05-17

### 改進
- 新增四項端對端自動化測試，涵蓋 JSON 結構驗證、概率歸一化、步驟累積寫入、概似度表格輸出、方向性結論句輸出
- 測試情境改為純資料格式（假說 + 先驗 + 證據），不再夾帶實作指令，確保測試的是技能本身的行為

## [0.5.0] - 2026-05-16

### 新增
- 新增 `live_mode` 開關，預設為 `false`
- 關閉時為純對話模式，不寫入任何檔案，適合一般社群使用者
- 開啟時（`live_mode: true`）才將概率狀態寫入 `bayes_state.json`，供視覺化 App 讀取
- 開關位於 SKILL.md 頂部的 `## 設定` 區塊，修改一行即可切換

## [0.4.0] - 2026-05-16

### 改進
- 每條證據處理後，直接在對話中顯示概似度表格（假說 / 概似度 / 理由）及更新後的後驗概率
- 格式精簡易讀，方便使用者和旁觀者即時理解每一步推理依據
- 移除可折疊的隱藏數學區塊，改為全程可見

## [0.3.0] - 2026-05-15

### 改進
- 對話過程中不再顯示貝氏數學運算表格，閱讀體驗更清爽
- 每條證據只輸出一句白話總結，非統計背景的使用者也能輕鬆跟上
- 完整數學計算仍在背後進行，並寫入 `bayes_state.json` 供視覺化展示
- 需要查看計算細節時，可展開隱藏區塊確認

## [0.2.0] - 2026-05-15

### 新增
- 每個階段新增易讀的總結表格，包含「調查前 / 現在 / 原因」三欄
- 最終結論表格同步採用白話格式，不暴露貝氏術語給使用者

## [0.1.0] - 2026-05-12

### 新增
- 首次發布 Live Demo 版本
- 每次貝氏更新後自動將概率狀態寫入 `bayes_state.json`，供即時視覺化 App 讀取
